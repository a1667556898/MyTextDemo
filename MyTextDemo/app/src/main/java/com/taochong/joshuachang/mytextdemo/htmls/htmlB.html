华地紫园

华地•紫园占地142亩，坐落于科学大道与海棠路、银杏路交口，重金跨国聘请美国UDG联创国际建筑大师、美国YPA景观设

计大师、世邦魏理仕担纲物管顾问，汲取数百年的新古典主义建筑精华，建筑13栋纯板式电梯洋房、10栋纯板式御景高层以及

专属俱乐部会所、国际名店商业街、专属国际幼儿园等完善配套，为城市顶级名仕量身定制一座30万平米高尚国际人文社区。
